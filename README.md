# Mapa do Bairro

## Instalação

* Clonar repositório:

  ```bash
  $ git clone git@github.com:aledruetta/mapa-bairro.git
  ```

* Instalar Yarn:

  [https://yarnpkg.com/en/docs/install](https://yarnpkg.com/en/docs/install)

* Instalar dependências:

  ```bash
  $ cd mapa-bairro
  $ yarn install
  $ yarn run dist
  ```

* Executar

  ```bash
  $ firefox dist/index.html &
  ```
